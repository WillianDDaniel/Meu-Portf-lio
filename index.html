Projetos

Personagens de Dragon Ball

Este é um projeto pessoal desenvolvido com Next.js, React e Tailwind CSS, apresentando informações e imagens dos personagens do anime Dragon Ball. O design é responsivo e construído como uma SPA (Single Page Application), com os dados sobre os personagens provenientes de uma API desenvolvida por mim.

Online Web Tools

Projeto construído utilizando HTML, CSS e JavaScript Puro. São 4 ferramentas, Sorteador de números, Conversor de Texto, Contador de Caracteres e Gerador de Senhas. Este um projeto pessoal, não utilizei nenhuma biblioteca, o site conta com design responsivo e um botão para alternar entre Dark e Light mode desenvolvido por mim.

API Personagens de Anime

Esta é uma API desenvolvida por mim utilizando Node.js e Express, que retorna imagens e informações sobres os personagens de Naruto e Dragon Ball. Está API foi construída para me servir os dados necessários, para construir os projetos referentes, aqui citados entre os demais projetos.
